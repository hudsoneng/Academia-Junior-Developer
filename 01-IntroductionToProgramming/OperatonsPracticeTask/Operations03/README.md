### Operations Practice Task

C# console exercise from my Developer Academy journey.

#### Summary

The application receives four numbers and calculates which number produces the lowest result after being divided by the remaining ones. The lowest result is then returned.

#### Technology stack

- C# / .NET Console Application

#### How to run
1. Open the project folder
2. Run:
   - `dotnet run`

#### Notes
- The current version uses user input values.
